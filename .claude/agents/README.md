# Agentes Especializados

Agentes são configurações de IA especializadas para tarefas específicas do projeto.

## Agentes disponíveis

Nenhum agente customizado ainda. Adicione arquivos `.md` nesta pasta conforme a necessidade.

## Agentes planejados

### `agente-dev.md`
Agente especializado em desenvolvimento de código para a stack React + Supabase + Vercel.

**Responsabilidades**:
- Criar componentes React com Tailwind
- Escrever queries Supabase com RLS
- Configurar rotas Next.js App Router
- Criar migrations de banco de dados

### `agente-produto.md`
Agente focado em definição e validação de produto.

**Responsabilidades**:
- Analisar ideias de SaaS e criar plano de MVP
- Sugerir funcionalidades baseadas no público-alvo
- Criar copy para landing pages
- Definir pricing e modelo de monetização

### `agente-marketing.md`
Agente para crescimento e aquisição de usuários.

**Responsabilidades**:
- Criar conteúdo para redes sociais
- Escrever emails de onboarding
- Sugerir estratégias de SEO
- Criar campanhas de cold outreach

## Como criar um agente

Crie um arquivo `nome-do-agente.md` com:

```markdown
# Nome do Agente

## Persona
Quem é este agente e qual seu papel.

## Especialidades
Lista de habilidades e conhecimentos.

## Restrições
O que este agente não deve fazer.

## Exemplos de uso
Casos de uso típicos.
```
