# Skills Customizados

Skills são comandos especializados que o Claude Code executa para tarefas repetitivas do projeto.

## Skills disponíveis

Nenhum skill customizado ainda. Adicione arquivos `.md` nesta pasta conforme a necessidade.

## Como criar um skill

Crie um arquivo `nome-do-skill.md` com:

```markdown
# Nome do Skill

## Descrição
O que este skill faz.

## Quando usar
Exemplos de quando invocar este skill.

## Passos
1. Passo 1
2. Passo 2
```

## Skills planejados

- `criar-saas.md` — guia passo a passo para iniciar novo SaaS do zero
- `configurar-auth.md` — configurar autenticação com Supabase em um projeto
- `setup-pagamentos.md` — integrar Stripe ou Mercado Pago
- `deploy-producao.md` — checklist completo antes de fazer deploy em produção
